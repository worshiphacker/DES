# DES
用python实现des加密算法的四种模式
main.py:运行该文件即可获取四种加密模式的密文以及加解密测试
des.py:DES算法的实现
read_write.py:读取args并将数据写入对应的txt文件中
random_data:随机文件的生成，需手动将生成的文件名其改为data.txt
tables.py：用于des加密的各种表
des_Cipher.txt：密文
des_iv.txt：初始化向量
des_key.txt：密钥
des_plain.txt：明文
detemp.txt：用于解密的数据
temp.txt：用于加密的数据
recording.txt：记录加解密测试的数据
